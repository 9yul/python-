#setInterval(함수 이름, 파일 이름(선택 사항), 반복 할 값, 반복할 시간)

현재까지 사용 방법은 3가지입니다.


1.setInterval("출력",""(출력에 변수 이름은 필요 없으므로 빈 값), "테스트", (반복할 시간))



2.setInterval("변수선언, "(변수 이름)", "(변수 값)", (반복할 시간))



3.setInterval("변수불러오기, "(변수이름)", "(변수 값은 필요없으니 빈값"), "(반복할 시간))
